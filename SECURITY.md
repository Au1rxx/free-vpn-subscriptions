# Security Policy

## Scope

This repository is public and should contain only public distribution artifacts, public docs, and sanitized status data.

For the current public deployment boundary, see:

- `docs/project-details.md`
- `docs/deployment.md`

## Do not disclose

Do not post:

- credentials or tokens
- SSH keys
- Terraform state
- private infrastructure details
- unsanitized control-plane state
- anything copied from the private control repository that is not intended for public release

## Reporting

If you discover a security issue in the public repository contents, open a private report through GitHub security reporting if available, or contact the maintainer without posting the sensitive details in Issues or Discussions.
