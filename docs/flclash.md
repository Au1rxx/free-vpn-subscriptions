# FlClash Guide

## Best Subscription Format

Use the Clash YAML subscription feed:

`https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/clash.yaml`

Backup release download:

`https://github.com/Au1rxx/free-vpn-subscriptions/releases/latest/download/clash.yaml`

## FlClash Setup

1. Open FlClash and go to profiles or subscriptions.
2. Create a profile from URL or add a remote subscription.
3. Paste the Clash YAML link above.
4. Import the profile and refresh it once manually.
5. Select a healthy node and connect.

## Why This Path Works Well

- Clash YAML is the most direct fit for FlClash users.
- The raw URL supports auto-refresh.
- The release asset gives you a snapshot download fallback.

## Common Issues

- If the profile loads but a node is slow, switch regions before changing global app settings.
- If the profile does not refresh, verify that FlClash saved it as a remote profile.
- If all nodes fail at once, compare with the public status dashboard first.
