# V2Ray Subscription Guide

## Direct V2Ray Subscription URL

Use this URL in V2Ray-compatible clients:

`https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/v2ray-base64.txt`

Backup release download:

`https://github.com/Au1rxx/free-vpn-subscriptions/releases/latest/download/v2ray-base64.txt`

## Common Clients

- v2rayNG
- NekoBox
- Hiddify Next
- Shadowrocket-compatible subscription import flows

## Import Steps

1. Open your V2Ray-compatible client.
2. Choose import from subscription URL.
3. Paste the URL above.
4. Refresh the subscription.
5. Select a node and test connectivity.

## Update Behavior

- The public V2Ray Base64 feed is refreshed every 6 hours.
- The live status page is refreshed hourly.
- Release snapshots provide stable downloadable assets.

## Troubleshooting

- If the client expects a Base64 subscription, use the URL exactly as published.
- If import succeeds but the connection fails, switch to another region.
- If multiple nodes fail at once, check the status feed before rotating local settings.
- If a client seems compatible but remote refresh fails, confirm it saved the URL as a subscription source and not a one-time import.
