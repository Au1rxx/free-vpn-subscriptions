# Clash Subscription Guide

## Direct Clash Subscription URL

Use this URL in Clash-compatible clients:

`https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/clash.yaml`

Backup release download:

`https://github.com/Au1rxx/free-vpn-subscriptions/releases/latest/download/clash.yaml`

## Who Should Use The Clash Feed

- Clash Verge Rev users
- FlClash users
- Clash Meta for Android users
- Mihomo-compatible clients that accept remote YAML profiles

## Import Steps

1. Open your Clash client.
2. Go to the profile or subscription section.
3. Add a new remote profile.
4. Paste the URL above.
5. Import the profile and refresh it once manually.
6. Select a node that looks healthy on the status page.

## Update Behavior

- The public Clash YAML is refreshed every 6 hours.
- The live node status feed is refreshed every hour.
- Release assets provide downloadable snapshot checkpoints.

## Troubleshooting

- If import fails, verify that the client supports remote YAML subscriptions.
- If a node connects slowly, refresh the profile and pick another region first.
- If all nodes fail, check the public status feed before changing local settings.
- If the profile does not refresh, make sure the app saved it as a remote subscription and not as a one-time local import.
