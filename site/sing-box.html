How to import the sing-box subscription

Use this guide if your sing-box client accepts a remote JSON configuration or subscription import flow.

Direct URL

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/singbox.json

Steps

  1. Open your sing-box client.
  2. Create a new remote configuration.
  3. Paste the direct URL above.
  4. Sync the profile.
  5. Select an outbound and test connectivity.

Troubleshooting

  • If your client only accepts local files, download the JSON and import it manually.
  • If routes look stale, refresh the remote configuration.
  • If imports work but traffic does not pass, compare with the live status feed.
