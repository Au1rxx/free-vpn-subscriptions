Mac decision path

  • If the app is Clash-oriented, start with the Clash YAML guide.
  • If the app is built around remote JSON profiles, start with the sing-box guide.
  • If the app expects a subscription URL, start with the V2Ray guide.

Before you troubleshoot

  • Confirm the app format first.
  • Use the raw source URL rather than a stale local file.
  • Check live status before changing local settings.
