How to use this feed in Hiddify Next

Hiddify Next is a multi-platform client with remote profile support and easy subscription import. Use the V2Ray feed for the simplest setup path.

Direct URL

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/v2ray-base64.txt

Steps

  1. Open Hiddify Next.
  2. Create a new profile or tap import.
  3. Choose import from URL.
  4. Paste the V2Ray feed.
  5. Wait for sync, then select a node and connect.
