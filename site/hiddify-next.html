Why use this path

  • Simple import flow for users who want a quick remote profile setup
  • Good fit when a client expects a standard subscription URL
  • Release snapshot available as a backup download path

Before troubleshooting

  • Check the live status dashboard first
  • Confirm the full subscription URL was saved in the profile
  • Refresh the profile after the first import completes

Steps

  1. Open Hiddify Next.
  2. Create a new profile or tap import.
  3. Choose import from URL.
  4. Paste the V2Ray feed.
  5. Wait for sync, then select a node and connect.

Common issues

  • If the app reports an empty profile, refresh the subscription and verify the public status page.
  • If import from URL fails, copy the link again and make sure the app is using the full subscription URL instead of a single-node link.
  • If some nodes work and others do not, switch regions instead of assuming the entire feed is down.
