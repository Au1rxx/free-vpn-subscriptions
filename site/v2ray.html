How to import the V2Ray subscription

Use this guide if your client expects a Base64 subscription link, including common Android and cross-platform V2Ray clients.

Direct URL

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/v2ray-base64.txt

Steps

  1. Open your V2Ray-compatible client.
  2. Select import from subscription URL.
  3. Paste the URL above.
  4. Refresh the subscription.
  5. Pick an available node and connect.

Troubleshooting

  • If import fails, verify that the client supports Base64 subscription URLs.
  • If the profile loads but a node times out, switch regions.
  • If all nodes fail at once, check the published status feed first.
