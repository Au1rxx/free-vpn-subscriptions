Shadowrocket not connecting: how to fix iPhone subscription and node problems

Use this page when Shadowrocket imports the subscription but still cannot connect, or when only some nodes seem to work on iPhone or iPad.

Most common reasons

  • The selected node is temporarily unhealthy.
  • The imported list is stale and needs a refresh.
  • The wrong import format was used for the exact flow.
  • Local app settings were changed before checking the feed itself.

Fastest checks first

  • Refresh the subscription list.
  • Switch to another region.
  • Compare the region with the live status page.
  • Confirm the original subscription URL is still saved.

What to do before changing iPhone settings

  1. Verify the list was imported from the recommended public URL.
  2. Refresh the subscription once manually.
  3. Switch to another node or region.
  4. Check the live status dashboard.
  5. Only then start considering app-specific configuration changes.
