Start here if

  • Your subscription imports but does not connect.
  • Your profile no longer refreshes.
  • Only some nodes work.
  • You are not sure whether the problem is local or remote.

First checks before anything else

  • Verify the client is using the correct format.
  • Confirm the saved source URL still matches the public feed.
  • Refresh the subscription manually once.
  • Check the live status page before changing broad app settings.
