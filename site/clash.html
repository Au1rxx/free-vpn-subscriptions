How to import the Clash subscription

Use this guide if your client imports YAML subscriptions such as Clash Verge Rev, FlClash, or other Mihomo-compatible tools.

Direct URL

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/clash.yaml

Steps

  1. Open your Clash-compatible client.
  2. Create a new remote profile or subscription.
  3. Paste the direct URL above.
  4. Refresh the profile after import completes.
  5. Choose a node with current active status.

Troubleshooting

  • If the profile fails to load, verify your client accepts remote YAML subscriptions.
  • If one region is slow, refresh and switch to a different node.
  • If all nodes fail, compare with the live status page before changing local settings.
