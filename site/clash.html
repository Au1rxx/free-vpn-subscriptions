How to import the Clash YAML feed

  1. Open your Clash-compatible client and go to the profile or subscription area.
  2. Create a new remote profile, subscription, or profile from URL entry.
  3. Paste the Clash subscription URL exactly as shown above.
  4. Import the profile, then trigger a manual refresh once the first sync completes.
  5. Choose a node that shows healthy status and apply your preferred routing mode.

Troubleshooting a Clash subscription that imports but does not connect

  • Make sure your client actually supports remote YAML subscriptions instead of single-node imports only.
  • Refresh the profile after the first import so the newest public nodes are pulled in.
  • If one region is slow, switch nodes before changing global app settings.
  • If every node fails at the same time, compare with the live status page before assuming your client is broken.
