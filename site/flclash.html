Best subscription URL for FlClash

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/clash.yaml

Why this format is the best fit

  • FlClash users usually want a Clash-style YAML subscription path.
  • The raw URL works well for auto-refresh behavior.
  • The release file is available if you want a snapshot download.

What to verify after import

  • The imported profile remains linked to the remote URL.
  • The node list refreshes after a manual sync.
  • The selected region matches a healthy entry on the status page.

Steps

  1. Open FlClash and go to profiles or subscriptions.
  2. Create a profile from URL or add a remote subscription.
  3. Paste the Clash YAML link shown above.
  4. Import the profile and refresh it once manually.
  5. Select a healthy node and connect.

Common issues

  • If the profile loads but a node is slow, switch regions before changing global network settings.
  • If the profile does not refresh, verify that FlClash saved the URL as a remote profile instead of a one-time import.
  • If all nodes fail at once, compare with the status dashboard first.
