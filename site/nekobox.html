Why use the V2Ray path here

  • It is the simplest broad public import route for this repository.
  • It works well when you want a single subscription URL for remote refresh.
  • It matches the mobile-first import style many users expect.

Recommended verification flow

  • Import the URL once and confirm the subscription group is preserved.
  • Refresh the group manually after the first import.
  • Use the public status page before treating a failed node as an app bug.

Steps

  1. Open NekoBox.
  2. Choose the import or subscription URL flow.
  3. Paste the V2Ray feed.
  4. Refresh the subscription group.
  5. Select a node and test latency.

Common issues

  • If import succeeds but rules are missing, remember that clients can prioritize node parsing over rule data depending on the source format.
  • If every node times out, check the public status page before changing app settings.
  • If the feed does not refresh, re-open the subscription URL flow and confirm the link was pasted without extra characters.
