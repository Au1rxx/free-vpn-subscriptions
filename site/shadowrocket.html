Why this path is recommended

  • The V2Ray feed is the safest default for broad client compatibility in this repo.
  • The raw URL supports refreshable subscription behavior.
  • The release asset gives you a snapshot fallback path.

Before troubleshooting

  • Confirm the full subscription URL was saved.
  • Refresh the imported list once after setup.
  • Check the status dashboard before changing app settings.

Steps

  1. Open Shadowrocket.
  2. Add a subscription from URL.
  3. Paste the V2Ray feed.
  4. Refresh the list.
  5. Select a node and connect.

Common issues

  • If a profile imports but the node does not connect, switch to a different region first.
  • If all nodes fail, compare with the public status dashboard before changing local client settings.
  • If your iOS import flow prefers another format, test the Clash feed as a fallback only after checking client compatibility.
