Primary V2Ray subscription URL

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/v2ray-base64.txt

Use this URL when

  • Your app imports Base64 subscription links.
  • You want a broad compatibility path across Android and other platforms.
  • You are using v2rayNG, NekoBox, Hiddify Next, or similar clients.

Do not use this URL when

  • Your app expects a Clash YAML profile.
  • Your app documents remote JSON profiles as the native import path.
  • You only need a one-time manual file import and prefer a release download.

How to use the V2Ray link correctly

  1. Open the app's subscription import or import from URL flow.
  2. Paste the V2Ray Base64 URL exactly as shown above.
  3. Save it as a remote subscription source.
  4. Refresh once manually to verify the source is reusable.
  5. Check the status dashboard if the chosen node does not connect.
