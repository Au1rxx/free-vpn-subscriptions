How to use this feed in v2rayNG

Use this guide when you want the simplest Android import path for the public V2Ray feed and need a subscription URL that can be refreshed over time.

Open V2Ray Base64 Download Latest Release Check Live Status

Best subscription URL for v2rayNG

https://raw.githubusercontent.com/Au1rxx/free-vpn-subscriptions/main/output/v2ray-base64.txt

Why this path is recommended

  • The V2Ray Base64 feed is the broadest fit for mobile clients in this repository.
  • The raw URL supports auto-refresh behavior.
  • The release asset gives you a fallback download path.

What to verify after import

  • The subscription remains linked to the remote URL.
  • The client can refresh the feed manually.
  • The selected node matches a healthy region on the status page.

Steps

  1. Open v2rayNG.
  2. Choose import from subscription URL.
  3. Paste the V2Ray feed.
  4. Refresh the subscription list.
  5. Select a node and test latency.

Common issues

  • If import fails, verify that you used the Base64 subscription link and not the Clash or sing-box feed.
  • If a node times out, switch to another region.
  • If every node times out, check the public status page first.
